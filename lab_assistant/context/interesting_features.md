# Interesting Features

Use this file as a compact memory of what tends to count as worth surfacing.

General rule:
- Prefer observations that change interpretation, suggest a mechanism, reveal a regime boundary, or motivate the next experiment.
- If new data exists, assume a plot is the highest-bandwidth way for the PI to assess it. Use words to frame what the plot shows, why it matters, and what remains uncertain.

Usually interesting:
- a feature that persists across a meaningful parameter sweep
- a sharp change in intensity, peak position, linewidth, or hysteresis behavior
- agreement or disagreement with an expected literature pattern
- a result that resolves a previous ambiguity
- a comparison that changes which observable seems trustworthy
- a plot that makes a new trend, null result, anomaly, or regime boundary visually obvious

Usually not enough on its own:
- file churn
- raw quantity counts
- a new plot that does not change the story
- an anomaly with no attempt to distinguish artifact from physics

Legacy lab-specific cues:
- For twisted transport summaries, Landau-fan-like structure in the relevant filling window can be more interesting than fixed-slice summaries.
- For optical datasets, interestingness often comes from a feature’s evolution across `n`, `D`, `B`, or polarization, not from a single isolated spectrum.
- For magnetic or RMCD-style measurements, coercive-field behavior, hysteresis evolution, and disappearance or persistence of contrast are often decision-relevant.
