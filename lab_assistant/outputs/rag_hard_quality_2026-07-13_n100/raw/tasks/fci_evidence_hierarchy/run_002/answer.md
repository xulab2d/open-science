**Inference**

Ranked for diagnosing a fractional Chern insulator:

1. **Thermodynamics + compressibility**: strongest bulk evidence. It rules out a purely edge/lead/contact-driven transport feature and supports a genuine incompressible correlated phase.  
2. **Transport**: essential for topology because it tests Hall/edge response. It rules out an ordinary trivial insulator if quantization/edge signatures are robust, but by itself may not rule out percolation, contacts, or inhomogeneous current paths.  
3. **Local probe**: tests whether bulk and edge features are spatially consistent. It rules out interpretations based solely on global averaging, macroscopic inhomogeneity, or unresolved edge reconstruction.  
4. **Optical/RMCD/PL**: useful as a filling-sensitive and magnetic/topological proxy. It rules out some nonmagnetic or filling-independent explanations, but is indirect unless tied to transport or compressibility.  
5. **Standalone compressibility**: identifies incompressible fractional states and rules out a compressible metal, but without transport/topological response it does not by itself establish Chern character.
