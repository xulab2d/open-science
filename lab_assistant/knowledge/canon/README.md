# Canon

These files are compact local-context notes the assistant should consult often.

Expected contents:
- collection overviews
- recurring project themes
- lab-history summaries
- plotting and figure-style reference maps
- internal pattern notes that do not belong in the broader field syntheses

Population rule:
- promote stable findings from `index/` and `extracted/`
- keep files short and high-signal
- prefer one file per collection or local theme over one file per document
