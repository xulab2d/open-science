# Reference Strategy

Use the knowledge base in three layers:

1. `syntheses/`
- first stop for topic-level understanding
- meant to hold compact, field-aware summaries that already assume baseline expertise

2. `papers/`
- expanding paper shelf with short result-centered annotations
- use when a response needs specific results, citations, or closer matching to an observation

3. `canon/`
- local collection notes, deck patterns, and lab-history context
- use when the question depends on how this lab works rather than only on what the field says

Compression priorities:
- recurring materials and phase spaces
- concrete result classes
- measurement logic that this lab actually uses
- a small number of strong syntheses backed by a wider paper shelf
