# Paper Shelf

Purpose:
- Maintain an expanding set of papers that are likely to matter again.

Style:
- organize by topic, not by year dump
- annotate each paper briefly
- capture the specific result that matters for this lab
- avoid long prose unless the paper is unusually central

What belongs here:
- flagship field papers
- Xu Lab papers and closely adjacent work
- papers repeatedly useful for interpreting current measurements
