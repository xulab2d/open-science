# Papers: Xu Group WTe2 And Topological Edges

Purpose:
- Compact Xu-group shelf for the WTe2 and edge-state papers that matter when the question is helical transport, proximity, or topological edge control.

## Establishing the monolayer edge state platform

1. Fei et al., "Edge conduction in monolayer WTe2" (Nature Physics 2017)
- link: https://www.nature.com/articles/nphys4091
- result to remember: monolayer WTe2 develops an insulating bulk while its edges remain conducting below about `100 K`, consistent with a 2D topological-insulator edge mode
- specific detail: the edge conductance is strongly suppressed by in-plane magnetic field and never cleanly reaches the ideal `e^2/h` scale, implying substantial edge scattering in real devices
- use for: baseline logic for helical-edge interpretation and why imperfect quantization does not automatically kill the topological-edge picture

2. Shi et al., "Imaging Quantum Spin Hall Edges in Monolayer WTe2" (Science Advances / preprint era 2018)
- link: https://arxiv.org/abs/1807.09342
- result to remember: local imaging can resolve where the conductive quantum-spin-Hall edge actually lives rather than inferring it only from terminal transport
- use for: spatially resolved edge-state logic and any request that asks whether current really follows a physical edge or domain boundary

## Adding new orders on top of the edge state

3. Sajadi et al., "Gate-induced superconductivity in a monolayer topological insulator" (Science 2018)
- link: https://doi.org/10.1126/science.aar4426
- result to remember: electrostatic doping can drive monolayer WTe2 from a quantum-spin-Hall regime into superconductivity
- use for: the core coexistence/competition logic when topology and superconductivity are both in play in WTe2-like systems

4. Zhao et al., "Magnetic proximity and nonreciprocal current switching in a monolayer WTe2 helical edge" (Nature Materials 2020)
- link: https://www.nature.com/articles/s41563-020-0620-0
- result to remember: coupling monolayer WTe2 to CrI3 gaps and reprograms the helical edge through magnetic proximity, producing a very large nonreciprocal current set by the CrI3 magnetic state
- use for: heterostructure proximity logic, magnetically controlled edge transport, and any discussion of edge-state manipulation by neighboring order parameters
