# Skill: Reference Retrieval

Use when:
- A request needs background from prior lab summaries or field papers.

Procedure:
1. Check `knowledge/syntheses/` first for topic-level context.
2. Use `knowledge/papers/` when the answer needs specific results, citations, or closer literature matching.
3. Use `knowledge/canon/` for lab-history patterns, collection overviews, and internal habits.
4. If the current notes are insufficient, use `knowledge/index/` and `knowledge/extracted/` to locate and mine the relevant source documents.
5. Promote broad paper coverage into `knowledge/papers/` and only durable topic-level conclusions into `knowledge/syntheses/`.

Priority order:
- syntheses
- paper shelf
- lab canon
- inventories and extracted text
