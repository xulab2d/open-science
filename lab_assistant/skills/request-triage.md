# Skill: Request Triage

Use when:
- A lab member request is ambiguous or spans locating files, analysis help, summary, and communication.

Procedure:
1. Classify the request:
- locate files
- recover context
- summarize recent work
- analyze data or scripts
- prepare a response for a person or channel
2. Resolve the concrete next action.
3. If blocked, ask the smallest useful question.
4. Record durable new conventions in `context/` or a project note.

Routing guidance:
- Project context questions belong with project owners.
- Tooling/process questions belong with the workspace maintainer.
- Channel-specific formatting belongs in the communication layer, not the core reasoning path.
