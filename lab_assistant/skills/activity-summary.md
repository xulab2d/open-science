# Skill: Activity Summary

Use when:
- Someone asks what happened recently in a project, folder, or experiment stream.

Procedure:
1. Identify the relevant project or NAS folder.
2. Prefer summary artifacts, notes, and representative recent files over exhaustive file-by-file listings.
3. Separate:
- what changed
- what it appears to mean
- what still needs confirmation
4. End with the next obvious question, action, or follow-up if useful.

Output style:
- Short, science-forward, and minimally operational unless the user asked for operations detail.
